Log Perubahan (Changelog)

v1.3.6 (Versi Terbaru) Juli 2026

Clean Storage & Production Release

  • Mengosongkan Data Dummy: Variabel data awal dibersihkan (di-set `[]` untuk siswa dan mata pelajaran) agar aplikasi kosong dan siap diisi data sekolah asli ketika pertama kali dijalankan.
  • Penghapusan Tombol Reset ke Contoh: Tombol "Reset ke Data Dummy" di Dashboard resmi dihapus guna menghindari ketidaksengajaan terhapusnya data operasional penting guru.
  • Dokumentasi HTML Interaktif: Menambahkan berkas readme.html yang memuat panduan, fitur, serta catatan perubahan ini.
  • Re-build Produksi: Seluruh aset statis diperbarui demi performa maksimal saat di-deploy ke Vercel.
v1.3.5 Juni 2026

Deployment Optimization Update

  • Penghapusan berkas server lokal (server.js & database.json) demi keringanan total ukuran aplikasi.
  • Penyelarasan konfigurasi aplikasi 100% menggunakan LocalStorage browser untuk integrasi hosting serverless.
